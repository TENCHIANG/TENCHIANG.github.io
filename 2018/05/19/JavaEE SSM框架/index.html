JavaEE SSM框架

发表于 2018-05-19 | 分类于 笔记 ， Java

Java EE的SSM框架是基于Spring的非常流行且上手快的轻量级企业框架，由Spring、Spring MVC、MyBatis组成。

  • Spring是一个轻量级的控制反转（IoC）和面向切面（AOP）的容器框架
  • Spring MVC 分离了控制器、模型对象、分派器以及处理程序对象的角色，这种分离让它们更容易进行定制
  • MyBatis 使用简单的 XML或注解用于配置和原始映射，将接口和 Java 的POJOs（Plain Old Java Objects，普通的 Java对象）映射成数据库中的记录

配置JspStudy

JspStudy提供Apache+Tomcat+MySQL环境。

  1. 安装JspStudy（在这里下载）
  2. 打开MySQL（JspStudy->其它选项菜单->服务管理器->MySQL->打开）

  3. 修改MySQL密码（JspStudy->其它选项菜单->MySQL工具->MySQL命令行）

  4. 插入数据表

配置jdk

最好是安装最新版的64位jdk（Eclipse同样也得是64位）。

  1. 下载jdk（在这里下载）
  2. 安装jdk
  3. 配置jdk（设置环境变量）
    • 配置CLASSPATH
    • 配置Path

配置Eclipse Java EE

  1. 下载Eclipse（在这里下载）
  2. 在eclipse配置server
    • 打开eclipse下面的Servers窗口，右键New->Server
    • 选择Apache->Apache Tomcat v8.0（JspStudy\tomcat里面的版本)
    • Tomcat installation directory为tomcat真实目录（如D:\JspStudy\tomcat）

导入SSM项目

  • 下载ssm.rar，解压至eclipse工作目录
  • eclipse->FIle->Open Projects from File System…
  • Directory…选择刚刚的ssm文件夹

运行SSM项目

  • 右键ssm项目->Run As->1 Run on Server
  • 稍等片刻，eclipse会出现一个浏览器的界面，地址是http://localhost:8080/ssm/
  • 此时还得在地址后面加入listCategory以调用方法，此时地址为http://localhost:8080/ssm/listCategory

Eclipse的基本操作

  • 新建Java Web项目
    • File->New->Dynamic Web Project
    • 输入Project name
    • 选择Target runtime为之前已经配置好的Apache Tomcat（没有的点击New Runtime…）
    • Dynamic web module version选择4.0（一般选择Server之后，这个会自动选择，无需特别修改）
    • 把依赖库解压并导入到ssm/WebContent/WEB-INF/lib（这一步是Java Web框架的步骤）
  • 新建包
    • 右键项目名->New->Package
  • 通过新建Class新建java文件
    • 右键包名->New->Class->Name输入java文件名（也是主类名）
  • 新建xml文件
    • 右键文件夹名（包就是文件夹）->New->Other…->Wizards输入xml选择XML->XML File
  • java ee项目热部署
    • tomcat\conf\context.xml\<Context reloadable=”true”>
