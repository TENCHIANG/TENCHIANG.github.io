激活汉化JetBrains

发表于 2018-05-23 | 分类于 笔记 ， IDE

JetBrains是一家捷克的软件开发公司，该公司位于捷克的布拉格，并在俄国的圣彼得堡及美国麻州波士顿都设有办公室，该公司最为人所熟知的产品是Java编程语言开发撰写时所用的集成开发环境：IntelliJ IDEA。
截至2017年6月，该公司工发布了24款开发工具与及相关产品，如：

  • AppCode - Swift 和 Objective-C IDE开发工具。
  • CLion - 跨平台的C/C++ IDE 开发工具，支持C++11 、C++14、libc++以及Boost。
  • DataGrip - 一款数据库客户端工具
  • GoLand - Go语言的集成开发环境。
  • IntelliJ IDEA - 2001年发布。一套智能的 Java 集成开发环境，特别专注与强调程序师的开发撰写效率提升。
  • PhpStorm - PHP IDE开发工具。
  • PyCharm - 一款结合了Django框架的Python IDE开发工具。
  • Rider - 一款快速，功能强大，跨平台的.NET IDE开发工具。
  • RubyMine - 一套强大的Ruby on Rails IDE开发工具。
  • WebStorm - JavaScript的开发工具。

其中很多产品都是口碑最好的收费IDE，如前端的WebStorm，PHP的PhpStorm，Java的IntelliJ IDEA，安卓的Android Studio（基于IntelliJ IDEA）。本文以前端开发环境JetBrains WebStorm进行说明。

安装

去官网下载并安装最新版的WebStorm，由于WebStorm不是免费版的，只支持30天免费试用，所以需要购买或者是进行激活。

激活

WebStorm于JetBrains的关系就好像eclipse-javascript与eclipse的关系一样，所以，激活WebStorm就相当于激活JetBrains。激活JetBrains有以下几种方式：

  1. 若资金允许请购买正版
  2. 通过学生认证激活
  3. 通过授权服务器激活

打开WebStorm->help->Register->Lincense server输入授权服务器的网址点Acticvate进行激活，有一些网友搭建的现成的授权服务器：

  • http://www.activejetbrains.gq
  • http://idea.imsxm.com

如果以上都被禁了的话，可以去网上再找一些试试，或者可以自行搭建授权服务器。

汉化

在网上下载对应版本的中文语言包resources_cn.jar，放入安装目录的\lib文件夹下，重启软件显示中文界面。

如果中文显示乱码

  • 移除resources_cn.jar重启软件还原成英文
  • 打开File->Settings->Appearance&Behavior->Appearance
  • 选中Override default fonts by(not recommended) Name:
  • 选择任意中文字体如Microsoft YaHei
  • 重新执行汉化步骤
