Jsp Servlet 环境搭建

发表于 2018-08-01 | 分类于 笔记 ， IDE

准备软件

  • JDK: jdk-10.0.2_windows-x64_bin 目录d:\jdk
  • Tomcat: apache-tomcat-9.0.10-windows-x64 目录d:\tomcat
  • IDE: eclipse-jee-photon-R-win32-x86_64 目录d:\Program Files\eclipse

环境变量

C:\Windows\System32\SystemPropertiesAdvanced.exe->环境变量

  • JAVA_HOME->D:\jdk
  • CATALINA_HOME->D:\tomcat
  • Path->%JAVA_HOME%\bin->%CATALINA_HOME%\bin

验证

  • java --version
  • javac --version
  • startup然后浏览器打开http://localhost:8080

IDE设置

修改eclipse根目录下的eclipse.ini文件

设置eclipse

  • Window->Preference->General->Workspace->Text file encoding->Other->UTF-8
  • Window->Preference->Server->Runtime Environment->Add...
  • Window->Preference->Web这下面的Encoding都选择UTF-8
  • Window->Preference->Appearance->Colors and Fonts->Text Font->Edit可以设置字体

运行Servlet程序

创建项目

  • File->New->Dynamic Web Project，设置Project name为HelloServlet，然后Finish
  • 展开左侧Project Explorer->HelloServlet->Java Resource，右键src，New->Servlet
  • 在弹出的Create Servlet窗口，输入Java package为yy5209zz.cn，Class name为HelloServlet，然后Next
  • 选择URL mapping中的HelloServlet，右边Edit，将Pattern改为/hello.view，OK->Finish

运行

右键左侧的HelloServlet.java->Run As->Run on Server，eclipse会打开自带的浏览器，在地址后面加上参数?name=xxx就可以看到效果了。

一旦运行服务器后，任何源代码的修改，只要保存了，eclipse就会自动编译部署，你只需要刷新浏览器就行，如果有错误则会在底部的Console体现出来，并且浏览器提示内部错误。

结束运行

在Console点击Terminate结束程序与服务器的运行。
